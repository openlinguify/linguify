Quick Start Guide

Get OpenLinguify running locally in minutes with this comprehensive guide. Perfect for new contributors and developers.

Prerequisites

Make sure you have Python 3.8+, Poetry, and PostgreSQL installed before starting.

1. Clone and Setup

1Clone the Repository

git clone https://github.com/openlinguify/linguify.git
cd linguify

2Backend Setup

cd backend
poetry install
poetry shell

3Database Setup

# Create database (PostgreSQL)
createdb linguify_db

# Run migrations
poetry run python manage.py migrate

# Load initial data (optional)
poetry run python manage.py loaddata fixtures/initial_data.json

4Static Files

poetry run python manage.py collectstatic --noinput

5Start Backend Server

poetry run python manage.py runserver
Backend Ready!

Your backend should now be running at http://localhost:8081

2. Portal Setup

1Portal Environment

In a new terminal window:

cd portal
poetry install
poetry shell

2Portal Database

poetry run python manage.py migrate

3Start Portal Server

poetry run python manage.py runserver 8001
Portal Ready!

Your portal should now be running at http://localhost:8001

3. Create Super User

cd backend
poetry run python manage.py createsuperuser

5. Development Workflow

Development Tips
  • Keep both servers running during development
  • Use poetry run python manage.py shell for Django shell
  • Run tests with poetry run pytest
  • Check code style with poetry run black .

Common Commands

Task Command
Run tests poetry run pytest
Format code poetry run black .
Update dependencies poetry update
Django shell poetry run python manage.py shell
Make migrations poetry run python manage.py makemigrations

6. Troubleshooting

Common Issues

If you get database connection errors:

  1. Ensure PostgreSQL is running
  2. Check database credentials in settings.py
  3. Create the database: createdb linguify_db

If Poetry is not working:

  1. Install Poetry: curl -sSL https://install.python-poetry.org | python3 -
  2. Add to PATH: export PATH="$HOME/.local/bin:$PATH"
  3. Restart terminal

If CSS/JS files are not loading:

  1. Run poetry run python manage.py collectstatic
  2. Check STATIC_URL and STATIC_ROOT in settings
  3. Ensure DEBUG = True for development
