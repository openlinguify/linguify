Linguify Command Palette Demo

Type / dans les zones de texte ci-dessous pour voir le menu de commandes en action.

Comment utiliser :
  • Tapez '/' au début d'une ligne ou après un espace
  • Utilisez les flèches haut/bas pour naviguer
  • Appuyez sur Entrée ou Tab pour sélectionner
  • Appuyez sur Échap pour fermer le menu
  • Tapez pour filtrer les commandes par nom ou description
