Comment utiliser :
  • Tapez du texte directement dans les blocs
  • Tapez / pour ouvrir le menu de commandes
  • Passez la souris sur un bloc pour voir les contrôles
  • Utilisez Entrée pour créer un nouveau bloc
  • Utilisez Backspace sur un bloc vide pour le supprimer
  • Dupliquez ou supprimez des blocs avec les boutons
