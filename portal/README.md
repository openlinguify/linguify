# OpenLinguify Portal

Landing page and public website for OpenLinguify - Open source educational apps platform.

## Features

- Multi-language support (EN, FR, ES, NL)
- App showcase pages
- Blog system
- SEO optimized
- Responsive design

## Technologies

- Django 5.x
- Bootstrap 5
- Internationalization (i18n)
- Gunicorn
- WhiteNoise
